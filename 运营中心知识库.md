## 🔑 核心业务流程

### 供应商入驻流程
```
1. 查看入驻公告 → 2. 提交入驻申请 → 3. 资质审核 → 4. 签署协议 → 5. 上架商品
```

### 采购流程
```
1. 采购需求立项 → 2. 发布采购公告 → 3. 供应商投标/报价 → 4. 评标/评审 → 5. 中标通知 → 6. 合同签订 → 7. 履约验收
```

### 运营立项流程
```
1. 项目立项申请 → 2. 可行性评估 → 3. 服务器规划 → 4. 部署实施 → 5. 运营推广 → 6. 数据监控
```
